# Бот для проведения викторины

Бот для проведения викторины для telegram и vk, задавать пользователю вопросы и проверять ответы.
## Окружение

### Как установить
```
python install -r requirements.txt
```

### Переменные окружения:
```
TG_BOT_TOKEN - токет телеграм бота
VK_ACCESS_TOKEN - токен сообщества в vk
QUIZ_FILEPATH - путь к локальному файлу или ссылка на файл с вопросами для викторины
```

### Запуск

#### Telegram
 ```
 python tg-bot.py
 ```

#### VK
```
python vk-bot.py
```
